# How to compile and run programs

## Week5

1. Open up the shell window
2. Move to week5 directory

```shell
cd week5
```

### TwentyOne

1. Complie the `TwentyOne.swift` program

```shell
swiftc TwentyOne.swift
```

2. Run the `TwentyOne` program and see the result
