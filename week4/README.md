# How to compile and run programs

## Week4

1. Open up the shell window
2. Move to week4 directory

```shell
cd week4
```

### Twelve

1. Complie the `Twelve.swift` program

```shell
swiftc Twelve.swift
```

2. Run the `Twelve` program and see the result

```shell
./Twelve ../pride-and-prejudice.txt
```
